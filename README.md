# Карта джерел Полтавщини

## для створення андроїд

```bash
# Установка
$ yarn install

# Запуск в режимі розробки
$ yarn dev

# Компиляція сервера
$ yarn build
# Запуск скомпільованого
$ yarn start

# генерація статичних сторінок
$ yarn generate

# Ініціалізація андроїд
$ yarn cap_init

# генерація андроїд програми
$ yarn cap_android
$ yarn cap_sync

# відкритя андроїд програми
$ yarn cap_open
```
